Daggit

Draw beautiful Git graphs

Features

  • Handles tricky graphs correctly [the main reason for its being].
  • Define the palette to match/contrast your style.
  • Choose sizing to suit the context.
  • No dependencies.
  • Concise, readable code.

Getting Started

  1. Build from the repo 
    
  git clone https://github.com/niczero/daggit.git
  cd daggit
  npm install
  npm run build:prod
  2. Inject your Git graph into a ul list
  3. Invoke Daggit
  4. Style to taste
